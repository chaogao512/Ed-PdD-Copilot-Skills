# Teacher Digital Competence Case: Figure Design

## Input Summary

为“区域中小学教师数字胜任力提升的组织支持机制研究”设计核心机制图。

## Skill Invoked

`governance-figure-designer`

## References Used

- `figure-types.md`
- `layout-patterns.md`
- `quality-audit.md`

## Main Output

## Figure Type

Teacher development support mechanism.

## Recommended Paradigm

四层闭环机制图：区域教育行政部门、学校管理层、教研共同体、教师课堂实践。

## Layout Sketch

```text
政策与资源供给层：区域教育行政部门
        ↓ 目标、资源、评价边界
学校组织支持层：校长/教务/信息化团队
        ↓ 时间保障、工具支持、制度激励
教研共同体层：教研员/学科组/同伴教师
        ↓ 共同备课、课例研修、反馈诊断
教师课堂实践层：教学设计、课堂实施、评价改进
        ↑ 课堂证据、反思日志、学生反馈
持续改进环：诊断 -> 支持 -> 实践 -> 反馈 -> 再设计
```

## Element and Label List

- “数字能力培训”改为“数字胜任力发展支持”。
- “平台使用数据”改为“课堂转化证据”。
- “教师评价”改为“发展性反馈与改进建议”。
- “管理监控”改为“组织支持与资源调配”。

## Caption Draft

图 X 区域教师数字胜任力提升的组织支持机制。该机制通过区域政策与资源供给、学校组织支持、教研共同体协作和教师课堂实践之间的反馈循环，推动数字能力从培训参与转化为教学改进行动。

## Quality Self-Check

- 教师不是被动培训对象，而是课堂实践和反思主体。
- 图中包含反馈循环，而不是单向管理链条。
- 图示区分了组织支持、教研协作和课堂转化证据。

## Remaining Gaps
