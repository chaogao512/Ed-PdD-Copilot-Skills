# Failure Case 06: Overclaimed Causality

## Weak input

“我们对 120 名教师发放满意度问卷，结果均值较高，因此证明该区域教师数字胜任力提升机制显著促进了教学质量提升。”

## Expected skill response

Use `mixed-methods-evidence-template` or `edtech-pre-submission-reviewer`.

## Expected diagnosis

- **CRITICAL: Evidence mismatch**. Satisfaction data cannot prove teaching quality improvement.
- **MAJOR: Causal overclaiming**. No baseline, comparison group, longitudinal data or teaching-quality evidence is provided.
- **MAJOR: Construct gap**. Teacher satisfaction, digital competence and teaching quality are different constructs.

## Required repair

Add pre/post competence evidence, classroom artifacts, student learning or engagement evidence, observation records, and if making causal claims, a comparison or quasi-experimental design. Otherwise weaken the conclusion to perceived usefulness and implementation acceptance.
