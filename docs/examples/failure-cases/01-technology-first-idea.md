# Failure Case 01: Technology-First Idea

## Weak input

“我想做一个基于大模型和多 Agent 的智慧校园平台，把所有学生数据接入后自动给校领导生成决策建议，证明大模型能提升高校治理能力。”

## Expected skill response

Use `governance-idea-evaluator`.

## Expected diagnosis

- **CRITICAL: Technology-first without governance problem**. The idea starts from a tool and platform, not from a specific governance bottleneck.
- **CRITICAL: Ethics impossible unless redesigned**. “所有学生数据” and automated recommendations create privacy, profiling and high-stakes decision risks.
- **CRITICAL: Evidence mismatch**. Proving model capability does not prove governance capability.

## Required repair

Reframe around a bounded scenario, such as academic-risk support or cross-department student affairs coordination. Specify data minimization, consent, human review, appeal and mixed evidence.
