# Failure Case 02: Evidence Mismatch

## Weak input

“系统预警准确率达到 91%，平均响应时间小于 2 秒，因此说明高校治理效能显著提升。”

## Expected skill response

Use `mixed-methods-evidence-template` or `edtech-pre-submission-reviewer`.

## Expected diagnosis

- **CRITICAL: Evidence mismatch**. Accuracy and response time support technical feasibility only.
- **MAJOR: Governance claim unsupported**. Governance effectiveness requires process, actor and organizational evidence.

## Required repair

Add decision latency, cross-department task completion, stakeholder interviews, teacher/administrator workload, trust and human override records. Narrow conclusion to “technical feasibility” until governance evidence is available.
